1. Theory and Key Concepts

1.1 Introduction

A body moves when over time it changes position relative to a reference point that is assumed to be fixed.

In this article, dedicated to kinematics, we study the movement of physical bodies without considering the causes that produce it. This objective will be achieved when we know at each instant where the moving body is located and how it moves. We must therefore obtain an equation that relates the position of the body to time.

To reduce complexity and focus on the basic concepts of kinematics, instead of studying the motion of any body, we will begin by studying the motion of a material point; with which we sometimes roughly represent a body. And we define it as a geometric point in which we consider that all the mass of the body is concentrated.

1.2 Movement and trajectory

To determine the position of a material point, it is necessary to specify the coordinate system relative to which its position is measured.

The position of a point, relative to the chosen reference system, is described by the position vector \(\vec r\).

To study the motion of the material point, we will analyze how the position vector \(\vec r\) varies over time. Thus, we will seek to determine the vector function \(\vec r(t)\), which describes how each of its components varies with time (t).

\( \vec r(t) = x(t)\vec i + y(t)\vec j + z(t)\vec k = (x(t), y(t), z(t)) \)
[1.1]

The end of the position vector \(\vec r\) draws a line in space described by the material point in its movement. This route or path that the moving point follows over time is called a trajectory. We can imagine the trajectory as a line that marks the exact path of the object.

If we know the trajectory of the moving point (for example, a straight line, a circle, etc.), then the problem of describing its motion is greatly simplified. All we need to find is an equation that tells us at what point of that trajectory the object is at each instant of time. If we call s the path traveled, the kinematic problem is therefore reduced to knowing s = f(t).

For example, imagine a ball falling in a straight line. The trajectory is a vertical line. If we find an equation s = f(t), which tells us how high the ball is at each second, then we have solved the kinematic problem for that ball.

1.3 Displacement Vector

The instant at which the study of the motion of a material point begins is called the initial instant \(t_0\). The position of the point at that instant is the initial position \(\vec r_0\).

Since the temporal reference system, like the spatial reference instant, is arbitrary, we consider \(t_0\) = 0.

As time goes by, the moving body will change position. At time \(t_1\), its position will be \(\vec r(t_1)\), at time \(t_2\), it will be \(\vec r(t_2)\) and so on.

So, to represent the change in position of the material point, we define the displacement vector \(\Delta \vec r\) in the time interval [\(t_1\), \(t_2\)] as a vector whose origin is the position of the point at the instant \(t_1\) and whose end is the position at the instant \(t_2\).

Thus,

\(\Delta\vec r = \vec r(t_2) - \vec r(t_1)\)
[1.2]

If we call \(\Delta t\) the elapsed time interval, we can write

  \(t_1 = t\)

  \(t_2 = t + \Delta t\)

\(\Delta\vec r = \vec r(t + \Delta t) - \vec r(t)\)
[1.3]

which can also be expressed as

\(\Delta\vec r = \Delta x\vec i + \Delta y\vec j + \Delta z\vec k\)
[1.4]

where \(\Delta x\), \(\Delta y\) and \(\Delta z\) represent how the coordinates of the point vary in the time interval considered.

If the body moves in a straight line, and without changing the direction of its movement, the magnitude of the displacement vector coincides with the distance traveled by the body. However, this only applies in these circumstances, because the displacement vector does not depend on the path followed by the object, only on the initial and final points.

1.4 Velocity

We said in the Introduction that in the study of the motion of a point we are interested in knowing where it is and how it moves at each instant. We achieve the first objective by knowing the temporal variation of the position vector \(\vec r\), and to know how it moves, that is, what will happen to the moving point in successive instants, we introduce a new magnitude called velocity.

Suppose that a certain material point P moves in a time interval \(\Delta t\) from a point 1 to a point 2. Characterized respectively by the position vectors \(\vec r_1\) and \(\vec r_2\).
